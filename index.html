Projects

Here are some of the projects I have created while studying at DevBootcamp.

  • Mirror Mirror

    Built and programmed a rasberry pi driven smart mirror as a team final project. The project was completed over a timespan of 7 days.

    Used a JavaScript based, open source platform, as the core driver.

    On top of the platform, we built all of the modules (features) for the mirror.

    We used IBM Watson, to allow the user to interact with the mirror through voice commands.

    For our custom modules, we used, Yahoo Weather API, News API, Google Maps API, MapQuest Traffic API, tambalAPI, Embedded Youtube/Facebook IFrame elements.

  • IMDB Clone

    As a team, we used the OMBD API to build a rails application that behaves similar to IMBD.

    This application uses bcrypt for user authentication and React Rails to make the front page load dynamically.

    Once registered, the user can search for movies, add movies to a watch list, 'favorite' movies, have conversations about movies, and has a profile that displays an activity feed of all of these features.

  • BORD

    Over two days, I built a website that allows a user to create an account and query Wikipedia using the MediaWiki API. Once the query was made, the results were saved to the user's history list.

    JQuery and AJAX were used for asynchronously updating the DOM and making API calls.
